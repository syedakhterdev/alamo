# angularjs-webpack

### Quick start

```bash
# clone our repo
$ git clone https://github.com/syedakhterdev/alamo my-alamo-app

# change directory to your app
$ cd my-alamo-app

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```

go to [http://localhost:8080](http://localhost:8080) in your browser.
